Web Development Questions and Answers

1. Mention four HTML5 structural tags and explain their roles in page layout.

<header>: Represents introductory content or a group of navigational links.

<nav>: Defines a set of navigation links.

<section>: Represents a thematic grouping of content, typically with a heading.

<footer>: Contains footer information for its nearest sectioning content or the entire page.

2. Why is the <section> tag important in HTML5? Provide a code example.

The <section> tag is important because it helps to semantically group related content.

<section>
    <h2>About Us</h2>
    <p>We are a company that values...</p>
</section>

3. Describe how <article> and <aside> differ in meaning and use.

<article>: Represents a self-contained piece of content that could be distributed independently.

<aside>: Contains content that is tangentially related to the content around it.

2. Compare the advantages and disadvantages of hosting videos locally vs externally (e.g., YouTube).

Advantages of Local Hosting:

  • Full control over the video content and quality.
  • No dependency on third-party services.

Disadvantages of Local Hosting:

  • Increased server load and bandwidth usage.
  • Requires more storage space on the server.

9. Describe H.264, Ogg Theora, and WebM formats. Which is most widely supported?

H.264: A widely used video compression format known for its high quality and efficiency.

Ogg Theora: An open video compression format that is less common but free to use.

WebM: A format designed for the web, providing good quality and efficient compression.

H.264 is the most widely supported format across browsers.

1. What is Responsive Web Design (RWD)? Why is it important today?

Responsive Web Design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. It is important today because it ensures a good user experience across all devices, from desktops to smartphones.

1. Write a JavaScript function to find the factorial of a number.

function factorial(n) {
    if (n === 0) return 1;
    return n * factorial(n - 1);
}

2. Create a function to highlight all <p> elements when the mouse hovers over any one of them.

const paragraphs = document.querySelectorAll('p');
paragraphs.forEach(p => {
    p.addEventListener('mouseover', () => {
        p.style.backgroundColor = 'yellow';
    });
    p.addEventListener('mouseout', () => {
        p.style.backgroundColor = '';
    });
});

3. What are JavaScript events? Explain onclick and onmouseover with code examples.

JavaScript events are actions that occur in the browser, such as clicks or mouse movements. The onclick event triggers when an element is clicked, while onmouseover triggers when the mouse hovers over an element.

<button onclick="alert('Button clicked!')">Click Me</button>
<p onmouseover=" alert('Mouse over paragraph!')">Hover over this paragraph.</p>

4. Write a JavaScript program to find whether a number is a palindrome.

function isPalindrome(num) {
    const str = num.toString();
    return str === str.split('').reverse().join('');
}

5. What is the role of event.preventDefault() in forms? Explain with a code snippet.

The event.preventDefault() method prevents the default action of an event from occurring, such as submitting a form. This is useful for validation before submission.

document.querySelector('form').addEventListener('submit', function(event) {
    event.preventDefault();
    // Validation logic here
});

9. Create a function in JavaScript to toggle the display of a hidden <div> on button click.

function toggleDiv() {
    const div = document.getElementById('hiddenDiv');
    div.style.display = div.style.display === 'none' ? 'block' : 'none';
}
