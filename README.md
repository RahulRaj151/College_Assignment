## Learning Path

The repository documents progression from:
1. **Foundation** (Semester 1): Core Java programming fundamentals
2. **Consolidation** (Semester 2): Building on fundamentals with practical applications
3. **Specialization** (Semesters 3-4): Advanced topics in data structures, web, and mobile development
4. **Diversification** (Semester 5): C++ and specialized programming paradigms
5. **Capstone** (Semester 6): Full-stack software development and networking

## Languages & Technologies Used

- **Java**: Semesters 1, 2, 5 (primary language for early coursework)
- **C++**: Semester 5 (advanced problem solving)
- **Web Technologies**: Semesters 3, 4, 6 (HTML, CSS, JavaScript, related frameworks)
- **Mobile Development**: Semester 4

## Navigation

Each semester and assignment folder contains organized code files ready for review, compilation, and execution. Subdirectories are clearly labeled for quick access to specific coursework.
